# 基于忆阻器真随机源与随机计算的高容错图像边缘检测 FPGA 系统

**项目定位**：新型器件 TRNG 数字后处理与感存算一体硬件闭环原型验证 

**核心技术栈**：Xilinx FPGA, Verilog HDL, 混合信号架构, 随机计算，TRNG 后处理

## 💡 项目背景与核心痛点
传统数字图像处理系统在太空辐射、强电磁干扰等极端环境下，对单比特翻转极其敏感，高位数据的偶然翻转会引发灾难性的像素突变；而作为安全 SoC、加密芯片核心基础 IP 的真随机数发生器（TRNG），传统方案同样存在明显短板 —— 算法伪随机源可预测、易破解，模拟热噪声 TRNG 则易受工频干扰，难以通过严苛的随机性测试。

本项目依托课题组已授权的忆阻器 TRNG 国家发明专利，提取氧化物忆阻器（RRAM）**微观银离子随机扩散产生的纯物理底噪作为真随机熵源**，结合高容错随机计算架构，将图像像素映射为概率比特流；在 FPGA 平台完成 TRNG 数字后处理、边缘检测算子映射与全链路闭环验证，既实现了天然免疫单比特翻转的高实时视觉边缘检测，也为新型真随机数 IP 的芯片化集成与场景落地提供了硬件原型支撑。

## ⚙️ 核心硬件架构与代码解析
本仓库包含了系统的全部 FPGA 数字逻辑控制与算子引擎代码，运行于 256MHz 核心时钟下，主要分为以下几个核心模块：

* **`top.v` (系统顶层枢纽)**
    * 完成 128 路比较器脉冲采集、UART 通信、PWM 阵列及边缘检测算子流水线的顶层时钟路由与实例化。
* **`PWM.v` (128路并行数模转换映射)**
    * 采用**影子寄存器（Shadow Register）**机制，避免占空比动态刷新时产生逻辑毛刺。
    * 配合底层硬件 RC 滤波，为 128 路外部比较器提供精准映射图像像素灰度的动态参考电压。
* **`UART_128.v` & `uart.v` (高鲁棒性通信链路)**
    * 底层采用标准 AXI4-Stream 总线协议实现高效数据流转。
    * 顶层内嵌 **Timeout 超时状态机（150,000 时钟周期阈值）**，有效防止因外界干扰丢包导致的下一帧数据错位，极大提升了串口帧同步的鲁棒性。
* **`XOR_MUX.v` (随机计算微积分引擎)**
    * 利用异或门（XOR）代替减法器进行交叉相减去偏，利用选通器（MUX）代替加法器完成降维融合。
    * 严谨的**跨时钟域同步（CDC）设计**：外部输入的 100KHz 选通信号经过两级打拍（`sel_d1`, `sel_d2`）同步到 256MHz 时钟域，彻底消除异步亚稳态风险。
* **`signal_monitor.v` (概率解码统计窗口)**
    * 构建 1024 周期的高速积分统计窗口，将微积分后抽象的 0/1 随机序列还原为确切的边缘特征灰度数值，并握手 `m_axis_tvalid` 输出。
* **`m_wave.v` (学术基准对照组)**
    * 独立开发的基于本原多项式（x²⁰ + x³ + 1）的 20 位 LFSR 伪随机发生器，用于生成具有均等 0/1 概率的千万级伪随机序列，为 NIST 真假随机测试提供严谨的对照基准。
